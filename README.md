# Motchiy Discordbot Oss
このプロジェクトは、Minecraftサーバーの開始と停止操作を簡単にするための管理ツールです。  
著者が個人的に便利だと感じる機能を含んでいます。  
Ubuntu LTS上のPython 3で構築されており、軽量で実用的な運用に最適化されています。

## 主な機能
- コマンドでMinecraftサーバーの開始・停止が可能
- Discordボット連携を前提に設計（拡張が容易）
- ローカル環境および手動サーバー運用に最適化

## セットアップ
### Botサーバー
```bash
source setup.sh --token={your_token}
```

### Discordサーバー
`admin`, `mod`, `staff`というロールを作成し，適切に割り当ててください。  
|ロール名|役割|説明|
|:--:|---:|:---|
|admin|管理者|全ての機能を使えます。|
|mod|モデレータ|大体の機能は使えますが，サーバーを乗っ取れるようなものだけを削除しています。|
|staff|スタッフ|そんなにできることはないです。ちょっと信頼されているくらい？|

## 起動
```bash
source run.sh
```

## 終了
`^c`を入力

## 使い方
コマンドの先頭には `!` を付けてください。  
利用可能なコマンド一覧は `!help` で確認できます。
